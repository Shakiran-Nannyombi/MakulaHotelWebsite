# Makula Lady Hotel

A modern, responsive website for **Makula Lady Hotel**, a luxurious sanctuary designed exclusively for women in Kampala, Uganda.

## Overview

This project is a static website built to showcase the hotel's amenities, vision, and services. It focuses on a clean, elegant design with a specific branding tailored towards women looking for comfort, privacy, and luxury.

## Features

*   **Responsive Design:** Fully responsive layout using CSS Grid and Flexbox, optimized for mobile, tablet, and desktop views.
*   **Video Backgrounds:** Engaging video backgrounds on the homepage to set the mood.
*   **Exclusive Services:** Showcasing specific services like professional photoshoots.
*   **Contact Form:** Functional-looking contact form and styled contact information cards.
*   **Theme Customization:** Custom color scheme (Purple & Indigo) to match the brand identity.

## Technologies Used

*   HTML5
*   CSS3 (Custom Styling, No Frameworks)
*   Font Awesome (Icons)

## Setup

Since this is a static website, you can simply open `index.html` in any web browser to view it. No build steps or dependencies are required.
